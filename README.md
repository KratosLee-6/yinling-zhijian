# 银龄指尖 👴📱

> 让爸妈跟上时代 —— 专为中老年人打造的手机软件使用指南

## 项目简介

**银龄指尖** 是一个面向中老年用户（55-75岁）的 H5 微站，以**手书漫画 + 极简文字**的方式，帮助中老年人学会使用日常刚需 APP。

### 核心特点

- 📖 **极简文字**：每步操作不超过15字，看得懂、记得住
- 🎨 **图解教程**：真实 APP 界面 + 手绘箭头标注
- 📱 **适老设计**：大字体、大图标、高对比度
- 🏠 **无需安装**：H5 微站，手机浏览器直接打开

## 覆盖软件

| 软件 | 核心教程 |
|------|----------|
| 微信 | 发消息、视频通话、发朋友圈、付款 |
| 淘宝 | 搜索商品、购物车、下单付款 |
| 抖音 | 刷视频、搜索、点赞收藏 |
| 高德地图 | 导航、打车 |
| 铁路12306 | 查询车次、购票 |
| 手机设置 | 连WiFi、清内存（分品牌：华为/小米/OPPO/vivo/苹果/三星） |
| 支付宝 | 付款、收款 |
| 美团 | 点外卖、买菜 |
| 人社局 | 查社保、预约挂号 |
| 交管12123 | 查违章、预约考试 |

## 使用方式

直接用手机浏览器打开 `index.html`，或部署到任意静态托管服务（GitHub Pages、Vercel 等）。
